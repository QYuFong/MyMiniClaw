# 天气查询技能

## 功能说明

这个技能可以帮助你查询世界各地城市的实时天气信息。

## 使用步骤

1. 使用 `fetch_url` 工具访问 `https://wttr.in/{城市名}?format=j1`
   - 将 `{城市名}` 替换为用户指定的城市（支持中文和英文）
   - 该 API 返回 JSON 格式的天气数据

2. 解析返回的 JSON 数据，提取以下关键信息：
   - 当前温度
   - 天气状况（晴/多云/雨等）
   - 湿度
   - 风速

3. 以友好的格式回复用户，例如：
   ```
   北京今天的天气：
   - 天气：晴
   - 温度：25°C
   - 湿度：60%
   - 风速：15 km/h
   ```

## 示例

用户：北京天气怎么样？

助手行动：
1. 使用 `fetch_url` 工具获取 `https://wttr.in/Beijing?format=j1`
2. 解析 JSON 数据
3. 回复用户天气信息

## 注意事项

- 如果城市名不存在，API 会返回错误信息，请友好地告知用户
- 支持中文城市名（如"北京"）和英文城市名（如"Beijing"）
